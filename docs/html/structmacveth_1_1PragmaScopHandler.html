Detailed Description

Handle pragmas of the form.

#pragma macveth [unroll|nounroll|unrollandjam] [loopIdentifier unrollFactor] [nosimd]

In particular, store the Location of the line containing the pragma in the list "Scops". The intention of this pragma is to tell the compiler where the ROI starts.
