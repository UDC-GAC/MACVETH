◆ VType

enum macveth::VectorIR::VType

Types of vector operations we distinguish according their scheduling.

Enumerator
MAP 

No dependencies between operations, so it can be used a unique vector operation.

REDUCE 

Reductions imply that the operations are sequential.

Nonetheless we use this type in order to detect these special cases which can be optimized

INDUCTION 

Inductions, e.g. A[i] = x.

SEQ 

In any other case, we just say that the vector operation must be sequential.
