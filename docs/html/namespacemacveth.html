◆ MVSIMDCostModel

enum macveth::MVSIMDCostModel

Costs models naming.

Enumerator
UNLIMITED 

Vectorize always, despite the not vectorized estimation cost.

CONSERVATIVE 

If the sequential code vs the SIMD-zed code has a better estimation, then MACVETH does not vectorize the code.

AGGRESSIVE 

Region to vectorize can have parts where vectorization provides better performance.
