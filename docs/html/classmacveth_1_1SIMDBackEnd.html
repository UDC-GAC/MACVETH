◆ SIMDType

enum macveth::SIMDBackEnd::SIMDType

Types of SIMD instructions.

Enumerator
VFUNC 

Custom function.

VPACK 

Load, pack memory addresses.

VBCAST 

Broadcast values.

VGATHER 

Gather values from memory.

VSET 

Set values.

VSTORE 

Store values in contiguous memory.

VSCATTER 

Store values in a scattered fashion.

VMUL 

Multiplication.

VADD 

Addition.

VSUB 

Substraction.

VDIV 

Division.

VMOD 

Modulo.

VPERM 

Permutation or shuffle.

VREDUC 

Reduce operation.

VSEQ 

Scalar or sequential operation.

VOPT 

If the SIMD instruction is the result of an optimization (e.g.

fuse multiply-accumulation) we will call it VOPT

INIT 

Initializing some value, for instance, in reductions.
